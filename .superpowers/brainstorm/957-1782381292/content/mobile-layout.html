A — 竖向堆叠 + 汉堡菜单

VNC 固定在顶部，Chat 居中，Debug 底部。左上角 ☰ 打开侧边抽屉切换会话。

优点

  • 三个面板同时可见
  • App 感强

缺点

  • 每个区域都较小

B — 顶部 Tab 切换

对话 / VNC / 调试 三个 Tab 切换，☰ 打开会话列表抽屉。

优点

  • 每个面板有完整空间
  • VNC 可全屏查看

缺点

  • 需切换才能看到 VNC

C — VNC顶部 + 返回按钮

VNC 固定在顶部，顶栏有返回按钮，点击进入会话列表全屏页。

优点

  • 导航最直观
  • VNC始终可见

缺点

  • 切换会话需两步
